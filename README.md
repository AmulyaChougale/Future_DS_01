#  Super Store Sales Dashboard

The Sales Dashboard provides an interactive overview of business performance using key sales metrics and visual insights. It highlights total revenue, total profit, and total quantity sold, enabling quick evaluation of overall performance.

The dashboard includes visualizations such as sales and profit trends by year, quarterly sales distribution, and category-wise and segment-wise performance (Consumer, Corporate, and Home Office). It also displays sales distribution by state using a map, along with comparisons of profit and sales across product categories like Furniture, Office Supplies, and Technology.

These insights help users analyze trends, identify top-performing regions and categories, and support data-driven business decisions.
